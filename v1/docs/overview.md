# The Exponential Machine

## Core Vision
- **Genre:** Incremental/clicker game.
- **Theme:** Evolution of computing from manual calculation to post-human AI.
- **Scale:** Exponential growth ending in universe-scale re-engineering.

## Starting Eras
Players select a starting point:
1. **The Genesis Load (Phase 0):** 1939. Manual calculation, human effort.
2. **The Silicon Boot (Phase 4):** 1970s. Personal Computers, digital resources.
3. **The Neural Spark (Phase 7):** 2020s. GenAI, data centers, global networks.

## The 11 Phases
- Phase 0: The Cryptographic Dawn (1939-1940) - Human calculators and mechanical aids used to break Enigma.
- Phase 1: The Cryptographic Age (1940s) - The development of the Bombe and electromechanical automation.
- **Phase 2:** Mainframe Era (1940s-1950s) - Vacuum tubes, room-sized computing.
- **Phase 3:** Integrated Circuit (1950s-1960s) - Silicon chips.
- **Phase 4:** PC Revolution (1970s-1980s) - Microprocessors, desktops.
- **Phase 5:** Internet Age (1990s-2000s) - Global networks, web.
- **Phase 6:** Mobile & Cloud (2000s-2010s) - Smartphones, data centers.
- **Phase 7:** GenAI Breakthrough (2020s-Present) - LLMs, reasoning.
- **Phase 8:** AGI Frontier - Sentient, autonomous systems.
- **Phase 9:** Embodied AI - Ubiquitous autonomous robotics.
- **Phase 10:** Post-Human Future - Neural-digital integration.
