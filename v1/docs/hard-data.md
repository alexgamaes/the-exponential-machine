# Hard Data: Personnel and Throughput (1935-1945)

## Personnel Scaling by Year
Estimated total headcount across Bletchley Park and outstations.

| Year | Total Personnel | Status |
| :--- | :--- | :--- |
| 1935 | ~50 | Pre-Start Context: Initial GC&CS staff. |
| 1939 | ~200 | **START:** WWII outbreak; move to Bletchley Park. |
| 1940 | ~1,200 | Deployment of the first Bombes. |
| 1941 | ~3,300 | Rapid expansion of naval Enigma decryption. |
| 1942 | ~5,000 | Hut 6/8 industrialization. |
| 1943 | ~7,000 | Peak of Bombe operations; shift to Colossus begins. |
| 1944 | ~9,000 | Full-scale industrial intelligence machine. |
| 1945 | ~10,000 | Absolute peak (January 1945). |
